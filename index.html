Facts about the World Wide Web and other Specific Browsers

  • The World Wide Web or WWW, was first created in 1990, and marks the start of the age of browsers.
  • In 1995, Microsoft released Internet Explorer. Which just so happens to be the main browser most people and companies use.
  • Later down the line when Windows 10 was released, Microsoft went on to release Microsoft Edge, which led to Internet Explorer shutting down for good.
